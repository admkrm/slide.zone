## CSS Gradients
## CSS Gradients - Nederlandse vertaling: kleurverloop, overgangen - Een gradient beschrijft de overgang van één kleur naar de andere - Deze overgang kan verlopen over meerdere tussenkleuren - We maken gradients met CSS notaties. De browser zet deze om tot een afbeelding - Afmetingen en verhoudingen nemen ze over van het element waarop de gradient wordt toegepast - We zetten ze op **background-image** - Twee soorten: _linear_ (lijn) en _radial_ (vanuit cirkel/ellips) gradients
## Linear gradients: basisvoorbeeld ```css body { background-image: linear-gradient(90deg, #fcb045, #fd1d1d, #833ab4); height: 100vh; } ```
## Linear gradients - De kleurovergang volgt een lijn waarvan je de hoek kan bepalen - 0deg = opwaarts - 90deg = naar rechts - 180deg = neerwaarts - De helling hoeft geen rechte hoek te zijn: bvb 65deg kan ook - Ook gebruik van keywords is toegelaten: to top, to right, to bottom, to left, to top right, ... - De overgang kan indien gewenst lopen over meerdere tussenkleuren (_color stops_) - Alle kleurnotaties (inclusief transparantie) mogen worden gebruikt: red, #ffffff, rgba(254, 137, 2, 1.000), ...
## Linear gradients - In onderstaand voorbeeld verschijnt tussenkleur oranje op 25% - Je kan % gebruiken, maar ook elke andere lengtemaat (bvb rem of vw) - Indien er geen % wordt meegegeven worden de kleuren evenredig verdeeld - Eerste en laatste kleur hebben geen % nodig (= 0% en 100%) ```CSS div { background-image: linear-gradient(90deg, yellow, orange 25%, purple); } ```
## Linear gradients - Als je zoals hier eindigt op 50% zal de laatstgenoemde kleur de resterende 50% voor zijn rekening nemen ```CSS div { background-image: linear-gradient(54deg,red,orange,yellow,green,blue 50%); } ```
## linear-gradients: voorbeeld link met gradient - De aanpak in dit voorbeeld zorgt voor een subtiele gradient op een button die voor een 3D-effect zorgt ```CSS a { display: block; color: black; text-decoration: none; padding: 1.5rem; font-family: arial; font-size: 2rem; background-image: linear-gradient(0deg, #e2e2e2, #dbdbdb 50%, #d1d1d1 51%, #fefefe); text-align: center; border: .05rem solid lightgrey; border-radius: 1rem; } ```
## Radial gradients - Vertrekt vanaf een punt volgens een cirkel- of ellipspatroon - Standaard bevindt dat punt zich in het midden van het element - Deze gradient volgt een ellipspatroon als het bevattende element een rechthoek is - Is het bevattende element een vierkant dan volgt de gradient een cirkelpatroon - De notatie laat nog toe bijkomende zaken mee te geven: vorm, grootte en positie van de gradient ```css body { background-image: radial-gradient(yellow,limegreen); height: 100vh; } ```
## Radial gradients: shape - De vorm van het element: _circle_ of _ellipse_ - Geef je bvb _circle_ mee, dan vormt de gradient een cirkel - Zelfs indien het bevattende element een rechthoek is ```css div { background-image: radial-gradient(circle, yellow,limegreen); } ```
## Radial gradients: size - In lengte eenheden of percentages - Slaan op de straal van de gradient - 1 waarde meegegeven = gebruikt voor breedte en hoogte (= cirkel) - 2 waardes meegegeven = eerste is de horizontale, tweede is verticale ```css div { background-image: radial-gradient(20rem 8rem, aqua, green); } ```
## Radial gradients: position - Bepaalt de positie van het centrale startpunt - Standaard = center center - Zelfde waardes als bij background-position, maar voorafgegaan door "at" - at top left, at right bottom, at center left, ... ```css div { background-image: radial-gradient(at top left, yellow, orange 50%, purple); } ```
## Repetitieve gradients - Als je van je gradients repetitieve patronen wenst te maken: bvb voor streepjespatronen - Eindigt altijd op een lengtemaat met vaste afstand (bvb rem) - Je gebruikt hiervoor de notaties repeating-linear-gradient() en repeating-radial-gradient() ```css div { background-image: repeating-linear-gradient(180deg, white, silver 3rem); } ``` ```css div { background-image: repeating-linear-gradient(45deg, orange, orange 1.2rem, white 1.2rem, white 2.4rem); } ```
## Gradients: "Harde" kleurovergangen - Door met een kleur te eindigen op een bepaald punt en op hetzelfde punt onmiddellijk te herbeginnen met een volgende kleur ```css body { background-image: repeating-linear-gradient(yellow, yellow 5rem,limegreen 5rem, limegreen 10rem); height: 100vh; } ```
## Gecombineerde gradients - Je kan meerdere afbeeldingen toevoegen aan een element, en dus ook meerder gradients - Door met de kleur transparent te werken kan je subtiele overlappingen en zeer complexe patronen creëren ```css body { background-image: repeating-linear-gradient(90deg, transparent, transparent 10rem, red 10rem, red 20rem, transparent 20rem), repeating-linear-gradient(transparent, transparent 10rem, yellow 10rem, yellow 20rem, transparent 20rem) ; height: 100vh; } ```
## Gecombineerde gradients: patroon ```css body { height: 100vh; background: linear-gradient(135deg, red 25%, transparent 25%) 0px 0, linear-gradient(225deg, blue 25%, transparent 25%) 0px 0, linear-gradient(315deg, yellow 25%, transparent 25%) 0px 0px, linear-gradient(45deg, pink 25%, transparent 25%) 0px 0px; background-size: 200px 200px; background-color: white; } ```

## CSS Transitions
## Transitions & Transforms - Overgangen (transitions): zorgen ervoor dat een element vloeiend en gradueel overgaat van de ene staat naar de andere - Transformaties (transforms): bewegen of veranderen het uitzicht van een element - Beiden staan op zichzelf maar werken heel goed samen en laten toe om (eenvoudige) animaties te maken - Geven waardevolle interactie en visuele feedback aan je gebruikers - Belangrijk: overdrijf niet en hou het eenvoudig, subtiel en consistent
## Transitions - Zorgen ervoor dat een element vloeiend en gradueel overgaat van de startsituatie naar de andere (bvb. hover) - Zonder transition verandert een element onmiddellijk van staat - Voeg je echter een transition toe, dan neem je controle over de verandering
## Transitions: verplichte eigenschappen - Een transition heeft 2 verplichte eigenschappen: - transition-property (eigenschap) - transition-duration (duurtijd) ```css .mijndiv { transition-property: background-color; transition-duration: 0.2s; } ```
## transition-property - De transition-property specificeert de CSS-eigenschap waarop de transitie zal toegepast worden - Is verplicht - Je kan verwijzen naar één specifieke eigenschap (e.g., background-color of transform) of naar alle eigenschappen ```CSS .mijndiv { transition-property: all; } .mijnanderediv { transition-property: transform; } ```
## transition-duration - Is eveneens verplicht - Bepaalt de duurtijd van de transitie ```CSS .mijndiv { transition-property: all; transition-duration: 1s; } .mijnanderediv { transition-property: transform; transition-duration: 0.5s; } ```
## transition-timing-function - Optioneel. Laat toe een patroon te kiezen voor de transitie - Standaard keuze is ease: start traag, wordt sneller, om dan weer te vertragen naar het einde toe - Andere timing opties zijn: ease, linear, ease-in, ease-out, ease-in-out, step-start, step-end, steps, cubic-bezier(#,#,#,#) - Subtiele verschillen, maar de moeite waard om mee te experimenteren ```CSS .mijndiv { transition-property: all; transition-duration: 1s; transition-timing-function: linear; } ```
## Transition delay - Optioneel. Met deze eigenschap geef je aan wanneer je wenst te starten - Standaard: starten van zodra het event wordt getriggerd - Met transition delay kan je de start-timing echt zelf gaan bepalen ```CSS .mijndiv { transition-property: all; transition-duration: 1s; transition-timing-function: linear; transition-delay: 5s; } ```
## Verkorte notatie (alles op 1 regel) ```CSS div { transition: all 3s; } div { transition: all 3s ease-in-out; } div { transition: all 3s ease-in-out 1s; } ```
## Transforms
## Transforms - Transformaties: bewegen of veranderen het uitzicht van een element - We behandelen hier enkel 2D-transformaties. Maar er bestaan ook 3D-transformaties. - Kunnen gebeuren wanneer een element van staat verandert: bvb een hover - Je kan het ook gewoon gebruiken op de CSS-startsituatie van een element
## Scale - Met scale kan je de omvang van een element groter of kleiner maken - Waarde 2 maakt een element 2 keer zo groot als de oorspronkelijke grootte: scale(2) - Waarde 0.5 doet een element met de helft kleiner worden: scale(0.5) - Je kan ook enkel doen uitzetten/inkrimpen in breedte of hoogte: scaleX(2) of scaleY(5) - Je kan dit ook verkort noteren als volgt: scale(2,5) ```css div:hover { transform: scale(2); } ```
## Rotate - Met rotate draait een element aantal graden mee of tegen de klok in - Positieve waarde, bvb 90deg = met de klok mee - Negative waarde, bvb -90deg = tegen de klok in - 1080deg = 3 volledige rotaties ```CSS div { transition: transform 1s; } div:hover { transform: rotate(1080deg); } ```
## Translate - Translate beweegt een element naar links/rechts en naar boven/onder - Beweging wordt gebaseerd op parameters gegeven voor X-(horizontaal) en Y-(verticaal) assen - Een positive waarde beweegt naar rechts en onder - Een negative waarde beweegt naar links en boven ```CSS div { transition: transform 1s; } div:hover { transform: translate(20px, 20px); } ```
## Skew - Skew kantelt of trekt een element scheef in de richting van de X- en/of de Y-as - Een positieve X-waarde kantelt naar links, een negatieve X-waarde kantelt naar rechts - Een positieve Y-waarde kantelt het element naar onder, een negatieve Y-waarde kantelt naar boven - Met een verkorte notatie geef je in één keer X en Y-waarden mee - Skewing doet ook de kinderen van een element mee kantelen. Je kan dit tegengaan door de kinderen de tegenovergestelde waarden mee te geven
## Skew ```css div { transform: skewX(25deg); transform: skewY(10deg); transform: skew(25deg, 10deg); } ``` ```css div { transition: transform 1s; } div:hover { transform: skewX(-10deg); } ```
## transform-origin - Transform-origin werkt in tandem met de transformeigenschap en voeg je toe aan parent selector - Je zet er het punt mee van waaruit de transform begint. Is standaard midden van het element (center) - Om het punt te verplaatsen naar linksboven gebruik je 0% 0% of left top - Rechtsonder: 0% 100% of right bottom, enz... ```css div { transform-origin: left top; } div:hover { transform: rotate(720deg); } ```
## transforms combineren - Doe je door ze na elkaar te benoemen ```CSS div { transform: rotate(90deg) scale(2) translateY(-50%) translateX(50%); } ```

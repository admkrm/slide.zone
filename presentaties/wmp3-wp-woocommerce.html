## WooCommerce
## WooCommerce - WordPress plug-in - Eigendom van bedrijf achter WordPress (Automattic) - Laat toe WordPress om te bouwen tot een E-commerce store - Alles wat je als shop/verkoper nodig hebt zit er in, inclusief betaling, transport en zelfs berekenen belastingen - Ook WooCommerce is freemium. Wij gaan werken met het gratis gedeelte - Als je WooCommerce in de praktijk gebruikt, voor een echte shop, ga je wellicht nood hebben aan enkele betalende opties
## WooCommerce ![WordPress](../img/wmp3-woo-1.png) ##### https://woocommerce.com
## Waarom WooCommerce? - Gebouwd boven op WordPress - Alle goodies die we tot nu toe hebben gezien kunnen gebruikt worden binnen de E-commerce store - WooCommerce is zeer flexibel - Ook WooCommerce is open source - E-commerce + blog gaan goed tesamen - Developer community
## Waarom WooCommerce? - Verkoop van fysieke producten, downloadbare, abonnementen, lidmaatschappen,... - Betaalmogelijkheden: via payment gateways zoals Stripe, PayPal, Braintree, ... - Verzendopties: vast tarief, live kostenberekening (USPS, UPS, Fedex...) - Labels drukken

## Afbeeldingen toevoegen - Maak ze mooi, let op achtergrond (wit bvb) - Zorg voor eenduidigheid - Let op de bestandsnamen: beschrijf het product, geen spaties maar '-' - Heel belangrijk voor SEO - Zorg dat ALT-text, title, caption en description ook telkens ingevuld zijn
## Product gallery - Maak zelf een product gallery aan - Meerdere afbeeldingen mogelijk - Je kan de afbeeldingen van volgorde veranderen - Tip: plugin smart image resize plugin voor WooCommerce - Hoogkwalitatieve afbeeldingen zijn essentieel onderdeel voor e-commerce
## Werken met product data - 'Sale' price - Je kan een schedule of timing kiezen voor de 'Sale' - Inventory / SKU / Voorraad - Shipping / afmetingen / gewicht - Linked products / cross-selling (bijpassende items) / up-selling (duurdere producten) - 'You may also like', 'You may also be interested in' - Advanced / Purchase note / Menu order / Enable reviews
## Categorieën en tags - Onze producten worden er beter vindbaar door - Zelfs bij kleine winkel < 20 items toch nuttig (SEO) - > 20 essentieel - 'Product categories' > Add new category - Elk product: 1 categorie - Elk product: meerdere tags - Categories: must have, tags: can have, maar zeker indien grotere store - Tags: gebruik niet dezelfde woorden als voor categorieën (voor gebruikers en SEO)
## Productvariaties aanmaken - Product data: 'Variable product' - Variations tab wordt zichtbaar - Attributes: bvb kleur, selecteer 'used for variations' - Groen | Geel | Blauw | Oranje - | = shift + option + l
## Productvariaties aanmaken - Bij variations kan je elke variatie beschrijven - 1-dimensionele variatie, er zijn meerdere mogelijk - Verschillende prijs, afbeeldingen, enz => probeer het zelf uit - Zorg voor een standaardafbeelding - Bekijk het in de front-end
## Global attributes - Bruikbaar bij verschillende producten - Product > Attributes - Bvb 'Size' - Daarna: 'Configure terms' - Bvb 'small', 'medium', 'large', 'extra large'
## Global attributes - Daarna terug naar je product > attributes - Hier kan je size toevoegen - 'Used for variations' aanvinken - Bij values kan je de verschillende maten toevoegen - Als je daarna kijkt bij variations, dan zie je dat voor elke variatie ook maten beschikbaar zijn
## Digital products - PDF boek, training videos, ... - 'Virtual' markeren - Betekent: niet verzenden, dus shipping tab verdwijnt - 'Downloadable' markeren, extra eigenschappen hiervoor verschijnen
## Korte beschrijving toevoegen - Bovenaan bij het product zichtbaar op productpagina, dus belangrijk - Bij editpagina product, onderaan de pagina > 'Product short description' - Hou het beknopt - Denk er goed over na, zeer belangrijk, leidt vaak tot verkoop
## Producten groeperen en 'project bundles' - Product data > 'Grouped product' - Heeft zelf geen prijs, wordt een container voor andere producten - Bij 'Linked products' tab kan je dan de producten selecteren die in deze groep gaan zitten - Er bestaat ook een premium plugin: 'Product bundles'
## Subscription products - Voor producten die bvb maandelijks naar de klant gestuurd worden - Gaan we niet zelf doen - WooCommerce heeft hiervoor een Hoogkwalitatieve oplossing - Premium plugin: WooCommerce subscriptions (199usd voor een enkele site)
## Producten importeren - Er zijn dummy producten ingebouwd in WooCommerce - Je kan eventueel ook via Storefront te installeren, via optie 'install dummy products' - Admin > Tools > Import - 'WooCommerce Products (CSV)' > Run Importer - WooCommerce plugin downloaden > Openen > Sample data - Run the importer - Beter om eigen producten aan te maken
